***
## Install
* ```
  $ git clone https://github.com/Lapp-coder/cleansys
  ```
* For M1-chip:
  ```
  $ make setup-arm64  
  ```
* For Intel-chip:
  ```
  $ make setup-amd64
  ```
## Usage
### Just run this command
```
$ cleansys 
```

## Configuration
### The configuration file is located at $HOME/.config/cleansys/config.json
